Annoyed just now by find(1), which as far as I can see doesn't have any option to select files with timestamps within a particular range _specified by date_.

You can say '-mtime +1234 -a -mtime -1236' for something like "last modified between 1234 and 1236 days ago", but now you have to figure out exactly how many days ago the date was. I wanted to just type a pair of YYYY-MM-DD dates into find!

I ended up making two reference files with the datestamps I wanted ('touch -d YYYY-MM-DD filename'), and then using '-newer this -a ! -newer that'. Clunky, but still easier than getting the date arithmetic right.

(Motivation: I'd found a reference in an old email to a one-off thing I once did, and wanted to find the code I'd used to do it, if it was still lying around.)

EDIT: thanks @Darius for the answer, which turns out to be
find . -newermt YYYY-MM-D1 -a ! -newermt YYYY-MM-D2 -print

Life began as a network of spontaneously self-organizing chemical processes. It's amazing. It got started early: sugars, amino acids, and the nucleobases that make up DNA have been found in asteroids and meteorites. But what's exciting me today is this: how much started even before the Sun was fully formed!

Recently astronomers have been checking out V883 Orionis, a very young star that hasn't even begun fusion. It's still growing by accumulating gas from a disk that's swirling around it. They've found icy dust grains containing at least 17 organic chemicals in this disk!

It seems these chemicals get formed in cold conditions. But the infalling gas heats up and produces intense bursts of ultraviolet radiation. Astronomers had worried that these bursts would destroy the organic chemicals, forcing them to get created again later on. But now it looks like these bursts actually help catalyze new reactions, forming more chemicals!

“We recently found ethylene glycol could form by UV irradiation of ethanolamine, a molecule that was recently discovered in space,” said Tushar Suhasaria, coauthor of a new paper on this protostar. “This finding supports the idea that ethylene glycol could form in those environments but also in later stages of molecular evolution, where UV irradiation is dominant.”

For more, check out this:

• Abubakar M. A. Fadul, Kamber R. Schwarz, Tushar Suhasaria, Jenny K. Calahan, Jane Huang and Merel L. R. van 't Hoff, A deep search for ethylene glycol and glycolonitrile in the V883 Ori protoplanetary disk, https://arxiv.org/abs/2507.14905.
